## Jave2

Pro's 
* Good to use for beginners/intermediate programers.
* Cross platform use
* Ease of use for converting video's

Con's
* Limited use on the optinos for FFMPEG
* Slower updates since it can lag behind the FFMPEG releases

## JavaCPP Preset

Pro's
* Good for advanced developers
* High Performance
* AI/ video processing 

Con's
* Use of pointers
* low-level api and harder implementation documentation 

## JCodec

Pro's
* Lightweight tools
* Simplicity for deployment
* Good to use for all levels of developer

Con's
* Slower rendering time
* Lack of supported formats like FFMPEG

## VLCJ

Pro's 
* Has great playback potential
* Open source with lots of documentation to find
* Usable for streaming, subtitles, webcams, ... lots of tools

Con's
* Requires a package to run
* Implementation and dependencys can be harder to implement

## Video4j

Pro's
* Easy to implement for computer vision
* Clean API and good for motion detection, and AI video analyzing

Con's 
* Small community and harder to find documentation
* Smaller community so less tested and solution to problems/ examples
